This is a readme file of the Supplementary Materials of the publication titled: Closed-loop automatic gradient design for liquid chromatography using Bayesian Optimization.

Contents and description:
sampleA_list.csv contains the consitituents and concentrations of sample A.
RetentionParams_Section4.1.txt contains the retention parameters of the samples used in Section 4.1, column 1 contains lnk0 values, column 2 S values.
RetentionParams_Section4.5.txt contains the retention parameters of the samples used in Section 4.5, column 1 contains lnk0 values, column 2 S values.
Measurements directory and subdirectories contains visualisations of all the measurements with and without baseline correction of MegaMix (sample B) and MiniMix (sample A)
